# Ajuste — remoção de colaborador

- Adicionado botão **Remover** na aba **Colaboradores**.
- A ação solicita confirmação antes de remover.
- O colaborador removido deixa de aparecer na lista e não consegue mais entrar no sistema.
- As comandas e os relatórios antigos do colaborador permanecem preservados.
- O administrador não pode ser removido por essa opção.
